# OTP UWP
an UWP Platform OTP Authenticator

## Features
- TOTP authenticator
- Support Steam authenticator algorithm
- Advanced settings (digits, algorithms...)
- Custom Labels (Icon, Title, Description...)
- Data export (plain, JSON format)
